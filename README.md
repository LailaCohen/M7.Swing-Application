# Swing Application
In this assignment I learned how to use Swing applications with JButton and JTextField. I connected everything we learned, including arrays, data types, calculations, and more.
I understood how different programming concepts work together to create a complete application. I enjoyed seeing my project come to life.
### What I Did
* Created the main window
* Added layout, size, colors, and fonts
* Created a title label
* Added a list of items
* Created a checkbox list for food items
* Added buttons to calculate the total price
* Added a label to display the total price
* Made the buttons respond to clicks
* Calculated the total of selected items
* Started the program
### What I Learned
* Swing Applications
* JButton
* JTextField
* Arrays
* Data Types
* Event Handling
* Calculations
* GUI Design
